
# Feedback App

A feedback app made with React.js and Context API for state management.

## Demo

https://preeminent-pithivier-b6b642.netlify.app/
## Installation

Install feedback-app-ui with npm

```bash
  cd feedback-app-ui
  npm install
```
    
## Usage/Examples

```bash
  npm start
```
